# PayloadForge — Reports Directory

This folder is automatically created at runtime to store JSON analysis reports generated by the framework.

## Report Format

Each report is saved as a JSON file with the naming convention:
```
report_<id>_<timestamp>.json
```

Example: `report_a1b2c3d4_20260530_104500.json`

## Contents of Each Report

- **report_id**: Unique 8-character identifier
- **generated_at**: UTC timestamp of report creation
- **disclaimer**: Legal/educational notice
- **session**: Full session data including:
  - `original_scan` — baseline signature scan result
  - `encodings` — all encoding results (Base64, XOR, ROT13, Hex, Multi-layer)
  - `obfuscations` — all 9 obfuscation technique results
  - `evasion` — comparative bypass rate analysis
- **analysis_summary**: Key metrics and defensive recommendations

## Note

Reports in this directory are excluded from git tracking via `.gitignore`.
To preserve specific reports, copy them to a separate location before committing.
